# 🚀 Roadmap Evolutiva: PiumaMD Next

Questa lista raccoglie idee e suggerimenti per le future versioni di PiumaMD, suddivise per categoria.

## 🎨 UI & UX (User Experience)
- [x] **Modalità Focus**: Una modalità a tutto schermo che nasconde ogni elemento della UI.
- [x] **Typewriter Scrolling**: Mantiene la riga corrente sempre al centro dello schermo.
- [x] **Multi-Window**: Supporto per aprire file diversi in finestre native separate.
- [x] **Mappa del Documento (Minimap)**: Anteprima del file a lato dell'editor.

## 🛠️ Funzionalità di Scrittura
- [x] **Wikilinks & Zettelkasten**: Supporto per `[[Nome File]]`.
- [x] **Ricerca Globale**: Cerca termini in tutti i file della cartella aperta.
- [x] **Smart Auto-complete**: Suggerimenti per titoli, tag e Wikilinks.
- [x] **Vim Mode**: Keybindings classici di Vim per power users.

## 📤 Esportazione & Integrazione
- [ ] **Esportazione Avanzata**: Aggiunta di formati come `.docx`, `.epub` o `.latex` tramite l'integrazione con **Pandoc**.
- [ ] **Cloud Sync**: Sincronizzazione automatica con Google Drive, Dropbox o WebDAV.
- [ ] **Pubblicazione Diretta**: Integrazione per pubblicare su Medium, Dev.to o Ghost.

## 🧩 Ecosistema
- [ ] **Sistema di Plugin**: Permettere agli utenti di estendere l'editor con plugin custom.
- [ ] **Mobile App**: Supporto per iOS e Android tramite Tauri v2.
- [ ] **Estensione Browser**: "Cattura" contenuti web in Markdown.
