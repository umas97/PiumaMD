# Benvenuto in PiumaMD 🖋️

PiumaMD è un editor Markdown di classe professionale, progettato per offrirti un'esperienza di scrittura fluida, potente e senza distrazioni. Questa guida ti aiuterà a scoprire tutte le funzionalità avanzate integrate.

---

## 🚀 Funzionalità Principali

### 📂 Gestione Workspace
Puoi aprire intere cartelle come workspace. La sidebar a sinistra ti permette di navigare agilmente tra i tuoi file. PiumaMD ricorda l'ultima cartella aperta e i file su cui stavi lavorando per farti riprendere sempre da dove avevi interrotto.

### 🧠 Sistema Zettelkasten (Wikilinks)
PiumaMD supporta i **Wikilinks** per collegare i tuoi pensieri.
- Scrivi `[[Nome File]]` per creare un collegamento a un altro file Markdown nel tuo workspace.
- Cliccando sul link nell'anteprima, il file corrispondente verrà aperto istantaneamente.

### 🧪 Matematica e Diagrammi
- **LaTeX**: Scrivi formule matematiche complesse usando la sintassi KaTeX. 
    - Esempio: `$E = mc^2$` o blocchi `$$ ... $$`.
- **Mermaid**: Crea diagrammi di flusso, sequenza o Gantt direttamente nel codice.
    - Esempio:
    ```mermaid
    graph LR
    A[Idea] --> B(Scrittura) --> C{PDF?}
    ```

## 📤 Esportazione Professionale

Offriamo tre modalità di esportazione per coprire ogni esigenza:

1. **Esportazione Nativa (PDF)**:
   - Accessibile da `File > Esporta in PDF (Nativo)`.
   - Genera PDF perfetti e leggeri con sfondo bianco garantito e layout ottimizzato per la stampa (senza tagli a metà pagina).

2. **Esportazione Avanzata (Pandoc)**:
   - Accessibile da `File > Esportazione Avanzata...`.
   - Permette di generare file **DOCX**, **EPUB**, **LaTeX** e **PDF Professionale**.
   - Il motore PDF avanzato utilizza **XeLaTeX**, garantendo il supporto perfetto per **Emoji** 🎮 e caratteri speciali.

3. **Esportazione HTML Standalone**:
   - Genera un file HTML completo di stili, font Google e supporto KaTeX, pronto per essere pubblicato o inviato via email.

## 🎨 Personalizzazione e Focus

### 🎭 Temi Material
Personalizza l'estetica dell'editor scegliendo tra diversi temi (Midnight, Dracula, Nord, GitHub Light, ecc.) dal menu `Visualizza > Cambia Tema`.

### 🧘 Focus Mode
Attiva la **Modalità Focus** (`Visualizza > Modalità Focus` o tasto `F12`) per nascondere sidebar, menu e status bar, lasciandoti solo con le tue parole.

---

## ⌨️ Scorciatoie Utili

- `Ctrl + S`: Salva il file corrente.
- `Ctrl + Shift + F`: Ricerca globale in tutto il workspace.
- `F12`: Attiva/Disattiva Modalità Focus.
- `ESC`: Chiude i pannelli di ricerca o esce dal Focus Mode.

---

### 🛡️ Sicurezza Dati
Non preoccuparti di perdere il lavoro:
- **Autosave**: Puoi attivarlo dal menu `Impostazioni`.
- **Conferma Chiusura**: Se provi a chiudere un tab con modifiche non salvate (indicate dal puntino ●), PiumaMD ti chiederà se vuoi salvare, scartare o annullare.
